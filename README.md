.dotfiles
=========

These are my dotfiles. You'll find my config for my favorite tools:
 - bash
 - vim
 - git
 - and more!

Dependencies
------------

 - [rcm](https://github.com/thoughtbot/rcm) is a management suite for dotfiles
 - [pack](https://github.com/maralla/pack) is a package manager for vim8

Install
-------

 - `rcup -v`
 - `pack install`
